# An Examination of Capital Punishment and its Evolution in the US

This exploratory analysis uncovers trends regarding the implementation of Capital Punishment in the US from the earliest European settlements until the present day. Primary techniques demonstrated throughout this project include data cleaning, exploratory data analysis, and visualization in Python.

## How to Use
1. Clone this repository.
2. Install the required Python packages:  
   pip install -r requirements.txt
3. Open the files in a Jupyter Notebook.

## Preliminary Results
The analysis has thus far shown, prior to dataset concatenation, that:
1. exceedingly more slaves have been executed than those in other professions
2. hanging was the most commonly used execution method, followed by electrocution and injection
3. and 1935 saw the most executions in one year (N=197)
